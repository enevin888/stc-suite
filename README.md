# SmartTech Engineering Review Suite

Expanded Streamlit QA/QC tool for BAS engineering documents.

## Review modules

- SOO Revision Review
- Points List Review
- Wiring Diagram Review
- Full Package Review

## Supported files

### SOO
- .md
- .txt
- .docx
- .pdf

### Points List
- .pdf table
- .xlsx
- .xls
- .csv

### Wiring
- .pdf
- .txt
- .dwg/.dxf upload warning only; export CAD to PDF first for reliable comparison

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
